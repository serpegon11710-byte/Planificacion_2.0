# Repository Language Standard

**Status:** Mandatory
**Scope:** Entire repository (files, docs, logs, code)

## Policy
1. **Monolingual Persistence:** All repository artifacts (code, documentation, governance, logs, READMEs) must be written exclusively in **English**.
2. **Naming Convention:** Filenames must be canonical (`README.md`, `index.md`, `policy.md`) and must not contain language suffixes (e.g., `.es.md`).
3. **No Duplication:** Multilingual copies of the same content are strictly prohibited.
4. **Interaction Bridge:** AI agents shall reason in the user's chat language, but must persist all outputs in English.
